# 现状功能清单(改造前快照)

> 记录时间:2026-06-30。本文件是 fullpage 整屏切换改造**之前**的项目状态快照,便于改造时对照、回退。

## 技术栈

| 项 | 选型 |
|---|---|
| 框架 | Vue3 + `<script setup>` + TypeScript |
| 构建 | Vite |
| 动画 | GSAP + ScrollTrigger |
| 平滑滚动 | Lenis(与 GSAP ticker 同步) |
| 内容 | 数据化,集中在 `src/data/*`,改内容不碰布局 |

## 交互模型(现状)

**长滚动单页**:整站是一条普通滚动文档,Lenis 接管滚轮做平滑滚动。各段落随滚动进度触发动画(scrub / ScrollTrigger reveal)。

> ⚠️ 这与目标不符。目标是**整屏切换**(滚一下直接翻到下一整屏),见 `02-整屏切换改造计划.md`。

## 目录结构

```
src/
  data/         ← 内容(改这里换成自己的)
    site.ts       品牌名 + 顶部导航(5 项:简历/方法/作品/目录/联系)
    profile.ts    首屏文案、简历、方法页 4 张卡(principles)
    works.ts      featured[](精选 5)+ catalog[](目录 9)+ catalogFilters
    contact.ts    联系页文案、二维码、社交链接
  sections/     ← 6 个段落组件
    HeroSection.vue
    ResumeSection.vue
    AboutSection.vue
    FeaturedSection.vue
    CatalogSection.vue
    ContactSection.vue
  components/
    SiteNav.vue        固定顶部导航,滚动高亮当前段,mix-blend-mode: difference
    ScrollProgress.vue 右侧圆形 SVG 进度环 + 百分比
    SectionFlash.vue   段落转场闪屏(scrub 驱动)
  composables/
    useSmoothScroll.ts Lenis 初始化 + scrollToId / scrollToTop(模块级单例)
    useReveal.ts       v-reveal 指令(ScrollTrigger onEnter 上移淡入,支持 data-reveal-stagger)
    useCountUp.ts      v-countup 指令(进入视口 0→目标值,1.6s power2.out)
  styles/tokens.css    设计 token + 基础 reset + Lenis 类 + .container
  main.ts              注册 v-reveal / v-countup 指令
```

## 各段落功能现状

### 1. HeroSection(首屏 #hero)
- 多镜头视频背景 `hero.videoSrc`(autoplay/muted/loop)。
- 入场:标题/副标题/按钮 `[data-anim]` 逐个上移淡入(stagger)。
- **滚动驱动(scrub)**:背景 Ken Burns 推近(scale→1.12);文案随滚动上移淡出退场(yPercent -22 + opacity 0)。
- CTA 按钮 `scrollToId(c.to)` 跳到目标段。

### 2. ResumeSection(简历 #resume)
- Bento 网格:左标题/简介/名片,右上数字统计,右中工作时间线,底部引言/技能/教育。
- 数字统计用 `v-countup`(进入视口时计数)。
- 统计块用 `v-reveal` + `data-reveal-stagger` 错位入场。

### 3. AboutSection(方法 #about)
- pin 舞台 + scrub 时间线(`end: +=200%`):
  - 背景 desk 图由近拉远(scale 1.25→1)。
  - 4 张卡(principles)随滚动依次浮入(stagger)。
  - **聚光态**(后半程):暗幕 spot-mask 压暗、非聚焦卡淡出、聚焦卡(index 2)放大。
- 已修过 `transform` 导致的层叠陷阱(`.cards` 不用 transform 居中)。

### 4. FeaturedSection(精选作品 #featured)
- flex 手风琴:折叠态竖排标题,展开态封面+详情。
- **自动轮播**:`setInterval` 2800ms 切换 activeIndex;hover 某张时 `focus(i)` 暂停;`@mouseleave` 恢复自动。

### 5. CatalogSection(作品目录 #catalog)
- 过滤 tab(catalogFilters)+ 响应式 computed 网格(3 列)。
- 卡片 `v-reveal` 进入视口上移淡入;hover 上浮 + 阴影。

### 6. ContactSection(联系 #contact)
- 视频背景 + 玻璃拟态联系卡 + 二维码手机框。
- 社交图标 hover 变绿(#16c784);mailto 链接;"联系我"跳首屏、"返回顶部"。

## 段落转场(SectionFlash)
- 全屏色板 0→peak→0,scrub 驱动,在 `App.vue` 配置 `trigger / color / peak`:
  - 首屏→简历:白闪 peak 0.95
  - →精选:暗入 peak 0.92
  - 目录→联系:暗叠化 peak 0.55

## 设计 Token(tokens.css)
- 强调色 `--accent: #9c60fc`(主紫)、`--accent-pink: #fca8e4`。
- 背景 `--bg-light: #f4f4f2`、`--bg-dark: #0e0e10`、`--grad-resume`(简历渐变)。
- 文字 `--text: #1a1a1a`、`--text-weak: #8a8a8a`、`--line: #eeeeee`。
- 尺寸 `--nav-h: 64px`、`--maxw: 1200px`。
